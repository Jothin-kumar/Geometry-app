
# Geometry app

A GUI application to solve geometry problems, Currently in beta stage.
To run it:
 - Run '''git clone https://github.com/Jothin-kumar/Geometry-app.git''' in terminal
 - Navigate to scripts folder
 - Run the "main.py" file
