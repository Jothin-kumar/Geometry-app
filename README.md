## To run geometry app:
### Linux:
```
git clone https://github.com/Jothin-kumar/Geometry-app.git
cd "Geometry-app"
pip install -r requirements.txt
cd scripts
python3 main.py
```
### Windows:
Note: Make sure that you have python3 installed.
```
git clone https://github.com/Jothin-kumar/Geometry-app.git
cd "Geometry-app"
pip install -r requirements.txt
cd scripts
python main.py
```
