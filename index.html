Geometry app

A GUI application to solve Geometrical problems!

Geometry app has a built-in diagram editor, in which you could draw diagrams as you would in a graph sheet!

Currently points, lines and angles(autodetect) are supported. Polygon support arriving in the near future! The current available features are:

  • Autodetect angles
  • Autodetect collinear points
  • Autodetect parallel lines
  • Create point automatically when two lines intersect
  • Want to add a feature? Visit github repository
